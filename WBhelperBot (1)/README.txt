1. Создай Telegram-бота через @BotFather и скопируй токен.
2. Перейди на https://render.com/ и войди через Google.
3. Нажми "New Web Service" → "Deploy from a Git repository" → "Manual Deploy"
4. Загрузи содержимое архива WBhelperBot.zip
5. В файле app.py замени YOUR_BOT_TOKEN на свой токен от BotFather.
6. Нажми "Deploy" — бот будет работать круглосуточно!

Чтобы бот отвечал на фото, просто отправь ему изображение товара.
